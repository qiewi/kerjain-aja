# **Simple TO:DO List - Project**  
_______

## Deskripsi
_______

Kerjain Aja is a TO:DO list project that allocates memory from local storage to store to-do(s) on personal level

## Features
_______

- Add TO:DO
- Delete TO:DO
- Check TO:DO
